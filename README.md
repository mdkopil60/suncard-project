☀️ SunCart – Summer Essentials Store

---

📌 Overview

SunCart is a summer-themed eCommerce web app where users can explore and purchase seasonal products like sunglasses, outfits, skincare, and more.

---

🚀 Features

- 🛍️ Browse products
- 🔥 Popular products section
- 🔒 Protected product details
- 🔐 Login & Register (Google + Email)
- 👤 User profile & update
- 📱 Fully responsive

---

🧱 Tech Stack

- Next.js
- Tailwind CSS + DaisyUI
- BetterAuth
- MongoDB

---

⚙️ Environment Variables
.env file

---

Run Locally

npm install
npm run dev
